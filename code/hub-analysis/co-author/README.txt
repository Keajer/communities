To get hubs:
1. Run hub.py 'full path of .net file' 'full path of corresponding .all.txt file'
*if you dont have .all.txt file go to the 'cluster-analysis/coauthor/' directory and follow the readme.txt there

For hub analysis:
1. Copy the 'allyears' directory under 'generic' directory of appropriate network branch. This directory contains the hublist file.
2. Run hub-analysis.py

For graphs:
1. Gun graph-generator.py
2. run makegraphs.r
